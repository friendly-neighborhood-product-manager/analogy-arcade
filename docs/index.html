Analogy Arcade

Pick a topic. Pick a vibe. Let AI turn “huh?” into “ohhh.”

Analogy Arcade is a browser-only AI learning app that explains confusing topics through worlds people already understand: football tactics, space missions, Bollywood drama, brick-by-brick builds, vacation itineraries, superhero-style teams, or any custom interest the user brings to the arcade.

What it does

A user enters a topic, chooses an interest world, selects an audience level, and gets a personalized learning card.

Input

  • Topic
  • Preset or custom interest world
  • Audience level
  • Output style

Output

  • Tiny definition
  • ELI5 explanation
  • Personalized analogy
  • Mapping table
  • Where the analogy breaks
  • Mini quiz

Agent workflow

The product is designed to show AI workflow thinking beyond a single prompt. 

User Input
  ↓
Explainer Agent
  ↓
Analogy Critic Agent
  ↓
Rewrite Agent
  ↓
Request / Output / Trace Logging
  ↓
Learning Card
Agent Role
Explainer Agent Creates the first learning card using the user’s topic, interest world, audience level, and requested style.
Analogy Critic Agent Scores the explanation for clarity, accuracy, analogy fit, usefulness, fun, and safety.
Rewrite Agent Improves the explanation using the critic’s feedback before the final answer is returned.

Why this exists

This project is intentionally built with free, browser-based tools to demonstrate practical AI Product Management skills: product framing, workflow orchestration, prompt design, quality evaluation, responsible AI guardrails, and public launch thinking.
